# Notes App

A simple and clean Notes application built with React.
It allows users to create, edit, and delete notes easily with a smooth user experience.

## Built With

* React
* Vite
* JavaScript

## Idea

The goal of this project is to practice building interactive user interfaces using React.
The app focuses on managing notes in a simple way where users can quickly add new notes, edit existing ones, or remove them.

## Features

* Add new notes
* Edit existing notes
* Delete notes
* Simple and clean UI
* Fast performance using Vite
